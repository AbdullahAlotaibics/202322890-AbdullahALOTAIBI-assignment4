# Assignment 4 — Personal Web Application

## Project Description
A polished personal portfolio web application built for Assignment 4. The site is presented as an interactive game inventory portfolio that highlights my game development, technical skills, projects, and contact information.

This portfolio showcases a unique theme while maintaining clarity and professional presentation, with interactive tabs, item details, API integration, and a hidden mini-game.

## How to Run Locally

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)
- Git installed on your machine

### Steps
1. **Clone the repository:**
    ```bash
    git clone https://github.com/AbdullahAlotaibics/202322890-AbdullahALOTAIBI-assignment4.git
    cd 202322890-AbdullahALOTAIBI-assignment4
    ```
2. **Open the project:**
    - Open `index.html` in your browser
    - Or use the Live Server extension in VS Code for live reloading

3. **View the website:**
    - The portfolio will load immediately
    - No build process or dependencies required
